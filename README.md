# Youtube-Video-Downloader :
Youtube-Video-Downloader est un projet open source pour convertir des vidéos YouTube en MP3 ou MP4. Libre d’utilisation et de modification, il est dédié uniquement à YouTube.

# Requirements :
> - Express ^4.18.2
> - Ffmpeg-static ^5.2.0
> - Fluent-ffmpeg ^2.1.2
> - Ytdl-core ^4.11.5
